# Assignment 12 – Web Component Library
Red River College Polytechnic  
Full Stack Web Development  

## Overview

This project is a React + TypeScript Component Library built using:

- Create React App
- Storybook
- Styled Components
- Jest + React Testing Library
- Docker (Production deployment)

The component library includes:

- Button
- Label
- Text
- Table
- Table Header
- Table Row
- Table Cell
- Table Footer
- Dropdown
- Radio Button
- Image
- Hero Image
- Card

All components:
- Are responsive
- Use Styled Components
- Have default and disabled states
- Change background color when disabled
- Include at least 2 tests
